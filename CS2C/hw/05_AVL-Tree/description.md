# Homework 5 AVL Tree
Implement an AVL Tree class which behaves similarly to `std::map`.

Any members or methods in the private sections can be modified however you like, as long as it's still in 
the spirit of the assignment. For example, you may use regular pointers instead of `std::unique_ptr` but you
can't use a `std::map` as part of your implementation.

Your methods must have the best possible big-O (e.g. balance_factor_() is O(1) and operator[] is O(log(n))). Grade Oven does not check this but I will deduct points if the big-O is 💩.
