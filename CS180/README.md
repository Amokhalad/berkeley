# CS180 — Intro to Computer Vision and Computational Photography

Consolidated project work for UC Berkeley CS180. Each subdirectory is a
self-contained project (originally a standalone repository).

| Project | Description |
| --- | --- |
| [RGB-Photo-Alignment](./RGB-Photo-Alignment) | Colorizing the Prokudin-Gorskii photo collection via channel alignment |
| [fun-with-frequencies](./fun-with-frequencies) | Filters, frequencies, hybrid images, and multiresolution blending |
| [face-morphing](./face-morphing) | Face morphing, the "mean face," and caricatures |
| [mosaicing](./mosaicing) | Image warping and panoramic mosaicing / stitching |
| [hdr-and-quilting](./hdr-and-quilting) | High dynamic range imaging and texture synthesis by quilting |
| [diffusion_models](./diffusion_models) | Diffusion models for image generation |
